ChatGPT Needs Specs: Using HWiNFO the Smart Way

Before ChatGPT can help you boost your FPS or fix weird performance bugs, it needs to know what it’s working with. That’s where HWiNFO comes in — a free tool that gives you a full snapshot of your system’s guts.

Step 1: Download and Launch HWiNFO

  1. Go to HWiNFO.com
  2. Choose the “Portable” version so you don’t have to install anything
  3. Run it and select “Summary-only” mode when prompted

You’ll get a clean overview of your CPU, GPU, RAM, motherboard, and more.

Step 2: Grab the Key Info

  • CPU (Model and Clock Speed)
  • GPU (Model and VRAM)
  • RAM (Total GB)
  • Storage Type (SSD or HDD)
  • Operating System

You can either type them manually or take a screenshot and use ChatGPT’s vision feature (if your version supports it).

Step 3: Build Your Prompt

My PC has a Ryzen 5 3600 CPU, 16GB of RAM, an RTX 3060 with 12GB VRAM, and a 1TB SSD running Windows 11. I get random FPS drops in Call of Duty Warzone. Can you help me optimize performance based on this setup?

Why It Works

  • It gives ChatGPT the full picture so it won’t suggest irrelevant fixes
  • It opens the door for real troubleshooting, not vague tips
  • It gives you better follow-up recommendations (drivers, settings, bottlenecks)

Once you’ve done this once, save the prompt or base template somewhere. You’ll reuse it a lot as we go deeper in the guide.
