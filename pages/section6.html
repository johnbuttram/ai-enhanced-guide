Section 6: Main Content Structure & Heading Hierarchy

Use a <main id="main"> element for the main content area of each page. This ensures accessibility and enables skip links to work correctly.

<main id="main" class="flex-1 p-6 bg-gray-900 text-white">
  <h1 class="text-3xl font-bold mb-4">Section 2: Topic Name</h1>
  <p class="mb-4">Introductory paragraph for this section...</p>
  <!-- page content, images, code blocks, etc. -->
</main>

Heading Levels

Start each page with an <h1> for the page title. Subsections should follow a logical heading order: h2, then h3, etc. Do not skip levels.

Content Styling

  • Wrap text in <p> and use mb-4 for spacing.
  • Use <ul>/<ol> for lists. Add list-disc list-inside if needed.
  • Use <blockquote> or styled <div> for important notes.
  • Avoid using .prose class to retain color control in dark mode.
  • Manually add Tailwind classes like font-bold to headings.

Images in Content

  • Always include alt text.
  • Use max-w-full h-auto to make images responsive.
  • Center images with mx-auto or use <figure> with <figcaption>.

Consistency is key: All pages should follow the same layout and formatting rules to maintain a professional and readable experience.
