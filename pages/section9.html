Section 9: Build Your Fix Kit

You’ve got the tools, the prompts, and the experience. Now let’s turn that into a real Fix Kit — something you can open up whenever a game runs like trash, your frames tank, or your system just isn’t feeling right.

What Goes in Your Fix Kit?

  • Prompt Packs: Bundles for diagnosing issues, optimizing settings, and cleaning up bloat
  • Favorite Tools: HWiNFO, MSI Afterburner, DDU, Revo, LatencyMon, etc.
  • Scripts: Batch or PowerShell tools you use regularly (ChatGPT can help generate them)
  • Prompt Templates: Custom ones you’ve refined through use

Prompt: Help Me Organize My Fix Kit

Help me organize a personal Fix Kit folder on my PC with subfolders for: - Tools - Prompt Packs - Batch Scripts - System Snapshots Include naming tips and ideas for maintaining it long-term.

Prompt: Generate a Printable Version

Take my system specs and most-used prompts and format them into a printable cheat sheet PDF. Add room for writing notes and tracking system changes.

Prompt: Build a Shared Version

Help me turn my Fix Kit into a shareable GitHub repo or ZIP pack. Include a readme that explains how to use it for other gamers.
Pro Tip: A Fix Kit isn’t just for solving problems — it’s for preventing them. Keep it updated, organized, and ready to deploy anytime.
